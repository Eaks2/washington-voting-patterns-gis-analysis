# Washington Voting Patterns GIS Analysis

GIS and spatial statistics project analyzing county-level voting patterns in Washington State using choropleth maps, spatial distribution measures, Moran’s I, Getis-Ord General G, and Local Moran’s I.

## Overview

This project analyzes Washington State county-level election data to identify geographic voting patterns, spatial clustering, and county-level outliers. The analysis uses election data joined with Washington county shapefiles to create map visualizations and spatial statistics outputs.

## Skills Demonstrated

- GIS analysis
- Choropleth mapping
- Spatial statistics
- County shapefile joins
- Spatial clustering and outlier analysis
- Data visualization
- Technical reporting
